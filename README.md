# paste-typer

## 使い方

1. 上のファイルをローカルに保存して paste-typer フォルダを作成。
1. Chrome を開き chrome://extensions/ にアクセス。
1. 右上の「デベロッパーモード」をオンにして、「パッケージ化されていない拡張機能を読み込む」で paste-typer フォルダを指定。
1. どのページでもテキスト入力欄（textarea, input, contenteditable）をフォーカスしてから Ctrl+V などで貼り付けると、拡張が貼り付け内容を取得して1文字ずつタイピングで挿入します。

## 動作確認済み

- MacBook Air M1, 2020, Chrome バージョン 141.0.7390.108（Official Build） （arm64）
